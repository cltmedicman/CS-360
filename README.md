  # Inventory Manager

  [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

  ## Description

  This is an android application for keeping track of stock inventory. This application has a user login screen, create new user screen, inventory list, ability to update, add new inventory, delete inventory and send an SMS message when the inventory drops below a certain threshold.
